# Decide 🗳️

*Libérez la voix de chacun dans vos décisions d'équipe !*

---

## C'est quoi, Decide ?

Imaginez un monde où **chaque voix compte** vraiment — pas juste les plus forts en réunion, mais tout le monde ! Decide est un outil qui permet à votre équipe de prendre des décisions ensemble, de manière fluide et inclusive.

Vous en avez assez des décisions prises par 2-3 personnes alors que l'équipe fait 20 ? Decide est fait pour vous ! 🎯

---

## ✨ Ce qui rend Decide unique

- **Participation de tous** — Même les plus discrets ont leur mot à dire
- **Processus guidé** — Pas besoin d'être un expert, Decide vous accompagne étape par étape
- **Asynchrone** — Pas besoin d'être tous ensemble au même moment
- **Transparent** — Tout le monde voit les propositions et peut voter
- **Simple & Minimaliste** — Fini les outils surchargés

---

## 🚀 Pour qui ?

- Les équipes projets qui veulent impliquer tout le monde
- Les associations et collectifs
- Les startup qui cherchent à prendre des décisions inclusives
- Toute équipe qui en a assez des réunions qui ne mènent nulle part

---

## 🛠️ Tech Stack

Ce projet utilise des technologies modernes :

- **Next.js 16** — Le framework React dernière génération
- **TypeScript** — Du code solide et bien typé
- **Prisma** — Base de données moderne avec PostgreSQL
- **Jest** — Tests pour dormir sur vos deux oreilles
- **Resend** — Pour les emails transactionnels
- **Jose** — JWT compatible Edge (Next.js everywhere!)

---

## 🏃‍♂️ Pour commencer

```bash
# Installer les dépendances
npm install

# Configurer la base de données
npx prisma db push

# Lancer le serveur de développement
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) et c'est parti !
